(Learning Checklist)🔹 বেসিক লেভেল (১ ঘণ্টা লাগবে)Headers: # থেকে ###### দিয়ে ছোট-বড় হেডিং তৈরি করা।Text Formatting: লেখাকে বোল্ড (**bold**), ইটালিক (*italic*), এবং স্ট্রাইকথ্রু (~~strike~~) করা।Lists: অর্ডারড (১, ২, ৩) এবং আন-অর্ডারড (* বা -) লিস্ট ও টাস্ক লিস্ট (- [ ]) তৈরি।Links & Images: অন্য ওয়েবসাইটের লিংক এবং নিজের রেপোজিটরির ছবি যুক্ত করা (![alt](url))।🔹 ইন্টারমিডিয়েট লেভেল (২-৩ ঘণ্টা লাগবে)Code Blocks: সাধারণ কোড ইনলাইন করা (যেমন: `docker run`) এবং ল্যাঙ্গুয়েজের নামসহ মাল্টি-লাইন কোড ব্লক তৈরি করা যাতে কোড রঙিন দেখায়।Tables: কলাম এবং রো তৈরি করে ডেটা সাজানো (যেমন: ডাটাবেজের তুলনা বা ফিচার লিস্ট)।Blockquotes: > চিহ্ন ব্যবহার করে কোনো বিশেষ উক্তি বা রেফারেন্স হাইলাইট করা।🔹 প্রো/অ্যাডভান্সড লেভেল (GitHub Specific)GitHub Flavored Markdown (GFM): গিটহাবের নিজস্ব স্পেশাল শর্টকাট।Alerts/Callouts: গিটহাবে চমৎকার নোট বা ওয়ার্নিং বক্স তৈরি করা (যেমন: > [!NOTE] বা > [!WARNING])।Mentions & Issues: কোডের ভেতর #12 লিখে গিটহাব ইস্যু লিংক করা বা @username দিয়ে কাউকে মেনশন করা।Emojis: লেখার সৌন্দর্য বাড়াতে ইমোজি কোড (যেমন: :rocket:, :warning:) ব্যবহার করা।Mermaid.js (ডায়াগ্রাম): কোনো এক্সটার্নাল ছবি ছাড়াই শুধু টেক্সট কোড লিখে ডাইনামিক আর্কিটেকচার ডায়াগ্রাম বা ফ্লোচার্ট তৈরি করা (সিস্টেম ডিজাইনের জন্য এটি সবচেয়ে শক্তিশালী টুল)।২. কোথা থেকে শিখবেন? (Best & Free Resources)অনলাইনে একদম ফ্রিতে প্র্যাকটিস করে শেখার জন্য সেরা ৩টি জায়গা:১. Markdown Tutorial (সবচেয়ে সেরা)লিংক: markdown-tutorial.comকেন ব্যবহার করবেন: এটি একটি ইন্টারঅ্যাক্টিভ ওয়েবসাইট। আপনি ব্রাউজারের একপাশে কোড লিখবেন এবং অন্যপাশে লাইভ আউটপুট দেখতে পাবেন। ১০ মিনিটে পুরো বেসিক শেষ হয়ে যাবে।২. GitHub Guides (অফিসিয়াল ডকুমেন্টেশন)লিংক: GitHub Mastering Markdownকেন ব্যবহার করবেন: যেহেতু আপনি গিটহাবে ওয়েবসাইট বানাচ্ছেন, তাই গিটহাবের নিজস্ব এই গাইডটি দেখলে অ্যালার্ট বক্স, টেবিল এবং গিটহাবের এক্সক্লুসিভ ট্রিকসগুলো নিখুঁতভাবে শিখতে পারবেন।৩. Markdown Cheatsheet (ঝটপট দেখার জন্য)লিংক: markdownguide.org/cheat-sheetকেন ব্যবহার করবেন: এটি বুকমার্ক করে রাখুন। লেখার সময় কোনো কোড ভুলে গেলে এখান থেকে ১ সেকেন্ডে দেখে নিতে পারবেন।৩. প্রো-টিপ: আপনার এডিটর সেটআপ (VS Code)আপনি যদি কম্পিউটার থেকে কোড লেখেন, তবে VS Code ব্যবহার করুন।VS Code-এ যেকোনো .md ফাইল ওপেন করে কিবোর্ডের Ctrl + Shift + V (ম্যাকে Cmd + Shift + V) চাপলে ডানপাশে ফাইলটির লাইভ প্রিভিউ দেখা যায়।গিটহাবে পুশ করার আগেই আপনার সাইটে লেখাটি কেমন দেখাবে, তা আপনি নিজের কম্পিউটারে বসেই দেখে নিতে পারবেন।
